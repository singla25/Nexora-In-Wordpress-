# 🚀 Nexora – Custom Social Networking Platform (WordPress Based) ## 📌 Project Overview **Nexora** is a custom-built social networking platform developed using **WordPress + Core PHP + AJAX**, focused on delivering a highly dynamic, scalable, and customizable user experience. Instead of relying on WordPress default systems, Nexora implements **custom authentication, networking logic, and AJAX-driven architecture**, giving full control over performance, scalability, and feature expansion. --- ## 🎯 Key Features ### 🔐 Advanced Authentication System * Fully custom user registration & login system * Secure session handling (without default WP auth) * Input validation & error handling * Forgot Password with OTP verification (Email-based) * Automatic login after password reset --- ### 🔑 OTP-Based Password Recovery * Email-based OTP verification system **Flow:** * User submits email/username * OTP sent via SMTP * OTP validation * Password reset * Scalable for future: * Email verification * Two-Factor Authentication (2FA) --- ### 💬 Chat System (Plugin-Based Integration) * Integrated **custom chat system using a modular plugin architecture** * Designed to work tightly with Nexora’s **connection system** #### ⚙️ Core Logic: * Only **connected users** can chat with each other * Chat list dynamically shows: * Accepted connections only * Active relationships #### 💡 Features: * One-to-one private messaging * AJAX-based messaging system * Near real-time updates (Polling / Long Polling in progress) * Clean messaging UI (mobile-style layout planned) * Message structure includes: * Sender / Receiver * Timestamp #### 🧠 Smart Restrictions: * ❌ No connection → No chat * ✅ Only mutual connections allowed * Prevents spam and enforces meaningful interactions #### 🗄️ Architecture: * Built as a **separate plugin module** * Custom database tables: * Messages * Conversations * Decoupled system → easy to scale and upgrade #### 🚧 Status: * Backend structure in development * Evaluating: * AJAX Polling * Long Polling * WebSockets (future upgrade) --- ### 🔔 Notification System * Real-time notification system for user interactions **Covers:** * Connection requests * Accepted / Rejected connections **Features:** * Notification badge * Popup-based UI * Seen / Unseen tracking **Logic Improvement:** * Sender actions do NOT mark notifications as read * Receiver controls read state --- ### 📧 Email Notification System * SMTP-based email notifications **Triggered on:** * New user registration **Features:** * Admin alerts * Reliable email delivery * Scalable for future alerts & reminders --- ### 👤 User Profile System * Dynamic profile dashboard * Sticky profile header: * Profile Image * Cover Image * Name & Username * AJAX-based tab navigation * Dynamic profile routing --- ### 📝 Dynamic Forms (AJAX Powered) * Real-time form submission * No page reload * Reusable architecture * Built for scalability --- ### 🤝 Connections System (Core Networking Feature) * Add / Remove / Manage connections **Handles states:** * Pending * Accepted * Rejected * Removed **Smart UI behavior based on:** * Guest users * Profile owner * Logged-in users --- ### 🔗 Mutual Connections System * Displays shared connections * Improves networking discovery --- ### 💬 Modal-Based Connection Management * Dynamic modals for: * New Connections * Requests * History * All Connections * Mutual Connections * Fully AJAX-powered --- ### 🌐 Routing & URL Management * Clean profile URLs * Custom routing logic * Handles edge cases (homepage conflicts, etc.) --- ### 🎨 Modern UI/UX * Responsive layout * Smooth AJAX interactions * Improved usability: * Show/Hide password * Interactive UI components --- ### 🗄️ Custom Data Architecture * Custom Post Types: * user_profile * user_connection * Optimized relational queries * Custom admin columns --- ### ⚙️ Admin Control Panel * Set default profile image * Set default cover image * Centralized configuration --- ### 🔒 Secure File Handling * Secure uploads * File validation * Access control --- ### ⚡ AJAX-Based
