# 🚀 Nexora – Custom Social Networking Platform (WordPress Based) ## 📌 Project Overview **Nexora** is a custom-built social networking platform developed using **WordPress + Core PHP + AJAX**, focused on delivering a highly dynamic, scalable, and customizable user experience. Instead of relying on WordPress default systems, Nexora implements **custom authentication, networking logic, and AJAX-driven architecture**, giving full control over performance, scalability, and feature expansion. --- ## 🎯 Key Features ### 🔐 Advanced Authentication System - Fully custom user registration & login system - Secure session handling (without default WP auth) - Input validation & error handling - Forgot Password with OTP verification (Email-based) - Automatic login after password reset --- ### 🔑 OTP-Based Password Recovery - Email-based OTP verification system - Secure flow: - User submits email/username - OTP sent via SMTP - OTP validation - Password reset - Built with scalability for future email verification & 2FA --- ### 👤 User Profile System - Dynamic profile dashboard - Sticky profile header with: - Profile Image - Cover Image - Name & Username - AJAX-powered tab navigation (no reloads) - Dynamic profile routing system --- ### 📝 Dynamic Forms (AJAX Powered) - Real-time form submission - No page refresh required - Reusable and scalable form structure - Backend designed for future extensibility --- ### 🤝 Connections System (Core Networking Feature) - Add / Remove / Manage connections - Smart UI behavior based on: - Guest users - Profile owner - Logged-in users - Handles connection states: - Pending - Accepted - Rejected - Removed --- ### 🔗 Mutual Connections System - Displays shared connections between users - Enhances networking discovery --- ### 💬 Modal-Based Connection Management - Dynamic modals for: - New Connections - Requests - History - All Connections - Mutual Connections - Clean UI with AJAX-based data loading --- ### 🌐 Routing & URL Management - Clean and dynamic profile URLs - Custom routing logic - Handles edge cases (homepage vs profile conflicts) --- ### 🎨 Modern UI/UX - Clean and responsive profile layout - Interactive connection interface - Improved UX with: - Show/Hide password toggle - Smooth AJAX interactions --- ### 🗄️ Custom Data Architecture - Custom Post Types: - user_profile - user_connection - Optimized relational queries - Custom admin columns: - User Name - Connection Status --- ### ⚙️ Admin Control Panel - Set default profile image - Set default cover image - Centralized configuration system --- ### 🔒 Secure File Handling - Secure media uploads - User-specific file restrictions - Validation against unauthorized access --- ### ⚡ AJAX-Based Architecture - Multiple AJAX endpoints - Fast and seamless user experience - No unnecessary page reloads --- ### 🐞 Debugging & Optimization -
